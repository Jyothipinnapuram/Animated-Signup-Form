How to Run This Project
Method 1: Open Directly
Download or clone the project.
Open index.html in a web browser (Chrome, Firefox, Edge, etc.).
The login form should display with animations.
Method 2: Using Live Server (Recommended)
Install VS Code (if not already installed).
Open the project folder in VS Code.
Install the Live Server extension (from Extensions Marketplace).
Right-click index.html and select "Open with Live Server".
